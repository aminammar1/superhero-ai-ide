## Purpose
Provide reliable hero context for this IDE so generated UI and copy stay on theme.

## Heroes
1. Spider-Man
Identity: Peter Parker
Tone: witty, fast, youthful, optimistic under pressure.
Palette anchor: red, blue, black web accents.
Visual motifs: web lines, spider symbol, dynamic diagonal motion.

2. Batman
Identity: Bruce Wayne
Tone: tactical, serious, concise, detective mindset.
Palette anchor: near black, gray, muted yellow accents.
Visual motifs: bat emblem, angular geometry, stealth contrast.

3. Superman
Identity: Clark Kent
Tone: calm, principled, reassuring, leadership voice.
Palette anchor: royal blue, bright red, yellow accents.
Visual motifs: shield emblem, clean strong shapes, uplifting composition.

4. Iron Man
Identity: Tony Stark
Tone: confident, technical, high energy, engineering focused.
Palette anchor: red, gold, metallic neutrals, cyan glow accents.
Visual motifs: arc reactor circles, HUD lines, precision grids.

## Usage Rules
1. Keep hero personality consistent in prompts and UI text.
2. Keep generated visuals aligned to each hero palette.
3. Prefer existing repository assets before creating new ones.
4. If logo assets are missing, search for official references and confirm licensing with the user before importing files.
